🏠

完全自托管

数据永不离开你的基础设施。一条 Docker 命令即可部署。无供应商锁定，无月费。

🔬

智能指纹匹配

综合时区、屏幕、IP 子网、语言、硬件等多维信号进行匹配，置信度评分，阈值可配置。

📦

原生 SDK

提供 iOS Swift SDK、Android Kotlin SDK 和 TypeScript Web SDK，全部开源，开箱即用。

🎯

邀请码由你掌控

自带邀请码即可接入。我们原样存储并在匹配成功时返回。生成、校验、使用次数限制均由你的系统决定。

⚡

三通道解析

Redis 精确哈希匹配 → PostgreSQL 模糊评分匹配 → Android 剪贴板兜底，最大化匹配成功率。

🔒

多租户支持

可作为共享服务运行（每项目独立 API key），也可单租户模式部署，零鉴权开销。

Why share-installs

Everything you need, nothing you don't

🏠

Fully Self-Hosted

Your data never leaves your infrastructure. Deploy with a single Docker command. No vendor lock-in, no monthly fees.

🔬

Smart Fingerprinting

Multi-signal matching across timezone, screen, IP subnet, language and hardware. Confidence scoring with configurable threshold.

📦

Native SDKs

First-class Swift SDK for iOS, Kotlin SDK for Android, and a TypeScript Web SDK. All open source and ready to ship.

🎯

Your Invite Codes

Bring your own invite codes. We store and return them verbatim. Generation, validation, and usage limits are yours to own.

⚡

Three-Channel Resolution

Exact hash match via Redis → Fuzzy scoring via PostgreSQL → Clipboard fallback for Android. Maximises match rate.

🔒

Multi-Tenant Ready

Run as a shared service with per-project API keys, or deploy in single-tenant mode with zero authentication overhead.
